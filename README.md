# 👨‍💻 Portafolio Personal - Victor Andrés Moreno Sierra

Este es mi portafolio personal donde presento mi trayectoria, proyectos, habilidades y certificaciones como **Ingeniero Electrónico**, **AI Engineer**, y **Software Engineer**. El portafolio está diseñado con un enfoque minimalista y oscuro (dark theme), acentuado con detalles de neón para reflejar la estética tecnológica de la inteligencia artificial y la ciberseguridad.

## 🚀 Tecnologías Utilizadas

- **HTML5 Semántico:** Estructuración limpia y accesible del contenido.
- **CSS3 (Custom Properties & Animations):** Diseño visual nativo (Glassmorphism), animaciones fluidas y colores personalizados (cyan y violet).
- **Bootstrap 5:** Sistema de grillas (`row`, `col-*`) utilizado para garantizar un diseño 100% responsivo y adaptativo en todos los dispositivos móviles y de escritorio.
- **JavaScript Vanilla:** Lógica para interacciones suaves en el scroll, animaciones del fondo (Neural Mesh Canvas), y comportamiento de la barra de navegación.

## 🎨 Características Principales

- **Diseño Responsivo:** Visualización perfecta en cualquier tamaño de pantalla, con especial atención a las secciones de experiencia y proyectos en móviles.
- **Fondo Interactivo (Neural Canvas):** Un lienzo animado en el *Hero* con nodos que simulan una red neuronal, reaccionando sutilmente al movimiento del ratón.
- **Navegación Fluida:** Sticky navbar con efecto "blur" y navegación *smooth scroll* que resalta la sección activa.
- **Secciones Destacadas:**
  - **Sobre mí:** Biografía y estadísticas.
  - **Stack Tecnológico:** Habilidades clasificadas por área (Lenguajes, IA, Web, Cloud, Ciberseguridad).
  - **Proyectos:** Presentación detallada de proyectos seleccionados con métricas de impacto y enlaces.
  - **Experiencia:** Línea de tiempo responsiva.
  - **GitHub:** Contribuciones Open Source y simulación visual del mapa de aportes.

## ⚙️ Cómo ejecutar el proyecto localmente

Dado que es un proyecto de frontend estático, no requiere instalación de dependencias pesadas ni configuraciones de servidor complejas.

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/victorMorenoInge/Portafolio.git
   ```
2. **Navegar a la carpeta del proyecto:**
   ```bash
   cd Portafolio
   ```
3. **Ejecutar en el navegador:**
   - Puedes simplemente abrir el archivo `index.html` en tu navegador web de preferencia (Chrome, Firefox, Edge, etc.).
   - Para una mejor experiencia (y si tienes VS Code), puedes instalar la extensión **Live Server** y abrir el proyecto dando clic derecho en `index.html` > *Open with Live Server*.
