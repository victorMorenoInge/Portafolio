Computer Vision · Accesibilidad

Comunicación Sin Barreras

Sistema de traducción de lenguaje de señas en tiempo real que opera completamente en el navegador, sin servidores externos. Detecta y clasifica gestos de mano mediante modelos de visión por computador y los convierte en texto con latencia mínima, garantizando privacidad total del usuario.

TensorFlow.js MediaPipe WebRTC JavaScript Computer Vision
↗ Ver demo ⌥ Repositorio

Problema

Las personas con discapacidad auditiva enfrentan barreras constantes de comunicación. Los sistemas existentes requieren conectividad y servidores externos, creando dependencias y riesgos de privacidad.

Arquitectura

Pipeline de inferencia on-device: captura de cámara → detección de keypoints con MediaPipe Hands → clasificación de gestos con red neuronal ligera → renderizado de texto en tiempo real.

Resultados

✓ Procesamiento en <50ms ✓ 0 datos enviados a servidores ✓ Historial + estadísticas offline

Impacto potencial

Escalable a kioscos, apps móviles y sistemas educativos. Aplicable en instituciones de salud, atención al cliente y plataformas de aprendizaje inclusivo.

Investigación · IA Generativa

Síntesis de Imágenes Médicas con IA Generativa

Investigación comparativa entre Redes Generativas Antagónicas (GANs) y Modelos de Difusión Probabilística (DDPMs) para la generación de imágenes médicas sintéticas. El estudio evalúa calidad, diversidad y utilidad diagnóstica de las imágenes producidas por cada arquitectura.

PyTorch GANs DDPMs Python FID Score SSIM
↗ Leer investigación ⌥ Repositorio

Contexto

Los datasets médicos son escasos por restricciones de privacidad (HIPAA/GDPR). Los modelos de diagnóstico asistido por IA necesitan datos de entrenamiento que no siempre están disponibles.

Metodología

Implementación y entrenamiento comparativo de GAN (DCGAN/StyleGAN) vs DDPM sobre datasets de imágenes radiológicas. Evaluación cuantitativa con FID, SSIM e inspección cualitativa por dominio.

Hallazgos clave

✓ DDPMs: mayor estabilidad en entrenamiento ✓ GANs: menor costo computacional ✓ Imágenes sintéticas viables para augmentación

Aplicación empresarial

Hospitales, startups de HealthTech e investigación farmacéutica pueden usar síntesis de datos para acelerar el desarrollo de modelos de diagnóstico sin comprometer privacidad de pacientes.

Web · Impacto Social

Ecoturismo Sostenible — Boyacá Histórico

Plataforma web full-stack para promover el ecoturismo sostenible en Boyacá, articulando biodiversidad, patrimonio histórico (Ruta Libertadora) y conservación ambiental en una experiencia digital accesible para turistas nacionales e internacionales.

React Node.js MongoDB APIs de Mapas Responsive Design
↗ Ver plataforma ⌥ Repositorio

Problema

El turismo en Boyacá carece de una plataforma digital moderna que integre ecosistemas naturales, historia y rutas sostenibles, limitando el acceso de viajeros y el impacto económico regional.

Solución técnica

Plataforma con mapas interactivos georeferenciados, sistema de rutas personalizadas, contenido educativo sobre biodiversidad y backend de gestión para operadores turísticos locales.

Impacto

✓ Difusión digital de patrimonio regional ✓ Apoyo a economía local sostenible ✓ Educación ambiental integrada
